HE30 SVG icon extraction
==========================

This package contains 25 SVG icons extracted from the supplied HTML fragment.

Notes:
- Vector path data and source attributes are preserved.
- An SVG namespace was added so each file is a standalone SVG document.
- Several icons use Chakra CSS custom properties:
  --chakra-colors-icon-strong
  --chakra-colors-icon-subtle
  --chakra-colors-icon-brand
- Open index.html for a preview with fallback preview colors.
